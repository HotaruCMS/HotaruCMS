Sidebar Plugin for Hotaru CMS
--------------------------------
Created by: Nick Ramsay

Description
-----------
Give your sidebar widget functionality. Reorder widgets and enable multiple sidebars.

Instructions
------------
1. Upload the "sidebar" folder to your plugins folder. Install it from Plugin Management in Admin.
