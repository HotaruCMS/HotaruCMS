Text Widget Plugin for Hotaru CMS
---------------------------------
Created by: Nick Ramsay

Description
-----------
Add multiple custom widgets to your sidebar. Paste text, HTML, PHP, etc into each widget and use the Sidebar Widgets plugin to position it in your sidebar.

Instructions
------------
1. Upload the "text_widget" folder to your plugins folder. Install it from Plugin Management in Admin.
