Search Plugin for Hotaru CMS
---------------------------------
Created by: Nick Ramsay

Description
-----------
Give your users the ability to search yor site.

Instructions
------------
1. Upload the "search" folder to your plugins folder. Install it from Plugin Management in Admin.
