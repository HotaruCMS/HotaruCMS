Categories Plugin for Hotaru CMS
--------------------------------
Created by: Nick Ramsay

Description
-----------
Add categories to posts, urls and the sidebar. Uses the Submit and Sidebar plugins.

Instructions
------------
1. Upload the "categories" folder to your plugins folder. Install it from Plugin Management in Admin.
