Pligg Importer Plugin for Hotaru CMS
--------------------------------
Created by: Nick Ramsay

Description
-----------
Import a Pligg database into Hotaru CMS. Version 0.1 imports the following tables: Links, Comments, Users, Categories, Tags and Votes. 

Instructions
------------
1. Upload the "pligg_importer" folder to your plugins folder. Install it from Plugin Management in Admin and follow the instructions.
