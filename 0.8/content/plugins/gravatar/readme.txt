Gravatar Plugin for Hotaru CMS
--------------------------------
Created by: Nick Ramsay

Description
-----------
Enable user avatars with Gravatar.

Instructions
------------
1. Upload the "gravatar" folder to your plugins folder. 
2. Install it from Plugin Management in Admin.
