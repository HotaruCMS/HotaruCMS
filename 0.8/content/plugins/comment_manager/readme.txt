Comment Manager Plugin for Hotaru CMS
--------------------------------------
Created by: Nick Ramsay

Description
-----------
Manage comments.

Instructions
------------
1. Upload the "comment_manager" folder to your plugins folder. Install it from Plugin Management in Admin.
2. Access it from the Admin sidebar under Plugin Settings.
