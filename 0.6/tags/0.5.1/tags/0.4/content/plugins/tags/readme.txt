Tags Plugin for Hotaru CMS
--------------------------------
Created by: Nick Ramsay

Description
-----------
Add tags to posts. Uses the Submit plugin.

Instructions
------------
1. Upload the "tags" folder to your plugins folder. Install it from Plugin Management in Admin.
