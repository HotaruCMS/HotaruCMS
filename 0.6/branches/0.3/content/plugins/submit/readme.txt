Submit Plugin for Hotaru CMS
----------------------------
Created by: Nick Ramsay

Description
-----------
Enable users to submit and display stories to your site. 

Instructions
------------
1. Upload the "submit" folder to your plugins folder. Install it from Plugin Management in Admin.
2. The index page for your site will show posts submitted using the "Submit a story" link in the navigation bar.
