Category Manager Plugin for Hotaru CMS
--------------------------------------
Created by: Nick Ramsay

Description
-----------
Total solution for creating and editing categories. 

Instructions
------------
1. Upload the "category_manager" folder to your plugins folder. Install it from Plugin Management in Admin.
2. Access it from the Admin sidebar under Plugin Settings.
