Installation
**************
To install this sitemap please do the following:
1. Unpack the plugin
2. Copy the sitemap folder to your "content/plugins" directory
3. Login to your Hotaru CMS and access the Admin section
4. Choose the Plugin Management section
5. Click the install button


How to Use
************
The sitemap plugin only has 3 options currently. To prepare for a sitemap:
1. From the Admin section choose Plugin Settings
2. Choose Sitemap
3. Enable or disable compression (disabled by default)
4. Set the frequency that pages are updated
5. Save your settings
6. Click Generate Sitemap
