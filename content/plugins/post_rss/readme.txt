Post RSS Plugin for Hotaru CMS
---------------------------------------------
Created by: Nick Ramsay

Description
-----------
This plugin enables RSS feeds for posts.

Instructions
------------
1. Upload the "post_rss" folder to your plugins folder. 
2. Install it from Plugin Management in Admin.
