Hello Universe Plugin for Hotaru CMS
------------------------------------
Created by: Nick Ramsay

Description
-----------
An example plugin showing how some features are made.

Instructions
------------
1. Upload the "hello_universe" folder to your plugins folder. Install it from Plugin Management in Admin.
