Widgets Plugin for Hotaru CMS
-------------------------------------
Created by: Nick Ramsay

Description
-----------
Give your site widget functionality. Reorder widgets and enable blocks to put your widgets in.

Instructions
------------
1. Upload the "widgets" folder to your plugins folder. Install it from Plugin Management in Admin.
