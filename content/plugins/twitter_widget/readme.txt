Twitter Widget Plugin for Hotaru CMS
---------------------------------
Created by: Jon Harvey
Uses: Twitterlibphp
Thanks to : Antonio Lupetti


Description
-----------
A sidebar plugin that displays your Twitter status, followers, etc.

Instructions
------------
1. Upload the "twitter_widget" folder to your plugins folder. Install it from Plugin Management in Admin.
2. Click on Twitter Widget and enter your Twitter username and password.
3. Turn widget on in the main widget settings panel in admin.
4. You can set up this widget to diplay your own tweets or your followers by editing twitter_widget.php,. See tutorial at http://woork.blogspot.com/2009/06/super-simple-way-to-work-with-twitter.html and the Twitter API Wiki.
