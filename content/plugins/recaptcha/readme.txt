ReCaptcha Plugin for Hotaru CMS
--------------------------------
Created by: Nick Ramsay

Description
-----------
The web's most popular anti-spam captcha system. By itself, it doesn't do anything, but other plugins can utilize it.

To use it, you'll need to register at http://recaptcha.net/ and get the necessary keys there.

Instructions
------------
1. Upload the "recaptcha" folder to your plugins folder. 
2. Install it from Plugin Management in Admin.
3. Edit settings in Admin -> ReCaptcha
