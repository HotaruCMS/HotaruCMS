Link Bar Example Plugin for Hotaru CMS
--------------------------------
Created by: Nick Ramsay 

Description
-----------
This plugin demonstrates how to hook into the Link Bar plugin and add an option to its settings page.

Instructions
------------
1. Upload the "link_bar_example" folder to your plugins folder.
2. Install it from Plugin Management in Admin.
3. Edit the settings in Admin -> Plugin Settings -> Link Bar
