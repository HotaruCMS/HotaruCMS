Vote Simple Plugin for Hotaru CMS
---------------------------------
Created by: Nick Ramsay

Description
-----------
Let users vote for posts. Adds pages for top, latest and upcoming stories as well as links for filtering top posts by time frame.

Instructions
------------
1. Upload the "vote_simple" folder to your plugins folder. Install it from Plugin Management in Admin.
