Media Select Plugin for Hotaru CMS
---------------------------------
Created by: Nick Ramsay

Description
-----------
This plugin adds a choice of "News", "Video" or "Image" to Submit Step 2, therefore tagging each post with a media type that can be used by other plugins. Media Selectalso adds a media choice to the category bar which allows users to filter posts to their choice of media.

Instructions
------------
1. Upload the "media_select" folder to your plugins folder. Install it from Plugin Management in Admin.
