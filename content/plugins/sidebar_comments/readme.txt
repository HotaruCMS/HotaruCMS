Sidebar Comments Plugin for Hotaru CMS
--------------------------------
Created by: Nick Ramsay

Description
-----------
Displays latest and top posts in the sidebar.

Instructions
------------
1. Upload the "sidebar_comments" folder to your plugins folder. Install it from Plugin Management in Admin.
