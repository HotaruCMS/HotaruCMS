Vote Plugin for Hotaru CMS
--------------------------------
Created by: Nick Ramsay

Description
-----------
Let users vote for posts. Adds latest and top story pages. Includes three different voting methods.

Instructions
------------
1. Upload the "vote" folder to your plugins folder. Install it from Plugin Management in Admin.
