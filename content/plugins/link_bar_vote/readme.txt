Link Bar Vote Plugin for Hotaru CMS
--------------------------------
Created by: Nick Ramsay

Description
-----------
Adds the ability to vote for a post directly from Link Bar.

Instructions
------------
1. Upload the "link_bar_vote" folder to your plugins folder.
2. Install it from Plugin Management in Admin.

Recommended
-----------
Move the CSS from the lb_vote_button.php template into your link_bar.css file.
