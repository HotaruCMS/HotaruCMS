Buzz It Plugin for Hotaru CMS
---------------------------------
Created by: Kyle Carlson

Description
-----------
This plugin adds a "Buzz This!" link below a post description, which when clicked, redirects you to your Google Reader account with the post's title & body in the body of the buzz.

Instructions
------------
1. Upload the "buzz_it" folder to your plugins folder.
2. Install it from Plugin Management in Admin.
3. Access it from the Admin sidebar
4. Save and view your site. A new "Buzz it!" link should be visible below each post.
