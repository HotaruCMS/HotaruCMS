Smilies Plugin for Hotaru CMS
---------------------------------
Created by: Nick Ramsay

Description
-----------
Converts text for smiley faces, e.g. :) into images. It uses the same images and text commands as Wordpress 
(http://codex.wordpress.org/Using_Smilies). Currently this plugin only applies smilies to comments and requires 
a commenting plugin.

Instructions
------------
1. Upload the "smilies" folder to your plugins folder. Install it from Plugin Management in Admin.
